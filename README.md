# phpBB 3.1 Extension - Hookup

## Installation

Clone into ext/gn36/hookup:

    git clone https://github.com/gn36/phpbb-ext-hookup ext/gn36/hookup

Go to "ACP" > "Customise" > "Extensions" and enable the "Hookup" extension.
